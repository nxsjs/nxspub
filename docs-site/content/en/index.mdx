# nxspub Documentation

`nxspub` is a release automation toolkit for multi-branch and workspace repositories.

It focuses on three things:

- predictable versioning from commit history
- branch-policy-driven release safety
- practical deploy orchestration with records and rollback

## Command Surface

```bash
nxspub version
nxspub release
nxspub deploy --plan
nxspub console --web
nxspub draft-doctor
nxspub git-hooks
nxspub lint --edit .git/COMMIT_EDITMSG
```
