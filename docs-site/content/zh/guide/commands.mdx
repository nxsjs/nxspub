# 命令

所有命令都支持 `--cwd <path>`。  
如果你是第一次用，按顺序看：`version -> release -> deploy -> console`。

## version（先用它）

<PackageManagerTabs>
  <PackageManagerTabs.Tab>

```bash
pnpm dlx nxspub version
pnpm dlx nxspub version --dry
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
npx nxspub version
npx nxspub version --dry
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
yarn dlx nxspub version
yarn dlx nxspub version --dry
```

  </PackageManagerTabs.Tab>
</PackageManagerTabs>

作用：

- 自动计算版本号
- 更新 changelog
- Git 提交 + 打 tag

## release（把刚才的版本真正发布出去）

<PackageManagerTabs>
  <PackageManagerTabs.Tab>

```bash
pnpm dlx nxspub release
pnpm dlx nxspub release --dry
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
npx nxspub release
npx nxspub release --dry
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
yarn dlx nxspub release
yarn dlx nxspub release --dry
```

  </PackageManagerTabs.Tab>
</PackageManagerTabs>

作用：

- 发布当前 `package.json` 的版本到 registry
- 常用先跑 `--dry` 预演
- 预发布分支常配合 `--tag alpha` 或 `--tag beta`

## deploy（部署到运行环境）

<PackageManagerTabs>
  <PackageManagerTabs.Tab>

```bash
pnpm dlx nxspub deploy --plan
pnpm dlx nxspub deploy --env production --strategy canary
pnpm dlx nxspub deploy --rollback --to <deploymentId>
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
npx nxspub deploy --plan
npx nxspub deploy --env production --strategy canary
npx nxspub deploy --rollback --to <deploymentId>
```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash
yarn dlx nxspub deploy --plan
yarn dlx nxspub deploy --env production --strategy canary
yarn dlx nxspub deploy --rollback --to <deploymentId>
```

  </PackageManagerTabs.Tab>
</PackageManagerTabs>

作用：

- `--plan` 只做计划与门禁检查，不执行真实部署
- 真实部署依赖 `deploy.provider` 配置（vercel/cloudflare/ssh 等）
- 回滚必须带 `--rollback --to <deploymentId>`

<PackageManagerTabs>
  <PackageManagerTabs.Tab>

```bash

```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash

```

  </PackageManagerTabs.Tab>
  <PackageManagerTabs.Tab>

```bash

```

  </PackageManagerTabs.Tab>
</PackageManagerTabs>

作用：

- Web 模式：`nxspub console --web --open`
- 严格只读审阅：`--readonly-strict`
- 用来给团队成员看清“将要发生什么”

## draft-doctor（草稿修复工具）

```bash
nxspub draft-doctor
nxspub draft-doctor --target 1.3.0
nxspub draft-doctor --target 1.3.0 --prune
```

## mcp（AI 客户端接入）

```bash
nxspub mcp
nxspub mcp init --client codex
nxspub mcp init --client cursor
```

## git-hooks / lint（提交前自动检查）

```bash
nxspub git-hooks
nxspub git-hooks --dry
nxspub lint --edit .git/COMMIT_EDITMSG
```
