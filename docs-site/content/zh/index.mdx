# nxspub 文档

`nxspub` 是面向多分支与 workspace 仓库的发布自动化工具。

它核心解决三件事：

- 基于提交历史自动计算版本
- 用分支策略约束错误发布
- 提供可回滚、可审计的部署流程

## 核心命令

```bash
nxspub version
nxspub release
nxspub deploy --plan
nxspub console --web
nxspub draft-doctor
nxspub git-hooks
nxspub lint --edit .git/COMMIT_EDITMSG
```
