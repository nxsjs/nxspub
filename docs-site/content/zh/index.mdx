# nxspub 文档

如果你第一次接触发布工具，按这份文档可以从 0 到 1 跑通一整套流程。

`nxspub` 主要做 4 件事：

- 根据 commit 自动算版本号
- 自动生成 changelog
- 执行 npm 包发布（release）
- 执行部署与回滚（deploy）

## 一条最短发布路径（可直接照做）

```bash
nxspub version
nxspub release
```

上面 2 条命令就能完成“版本升级 + 包发布”。

如果你还没准备上线部署，先不用看 `deploy`。
